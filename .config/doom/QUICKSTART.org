* Introduction

This is a personal Doom Emacs configuration focused on:
- *Org-mode* as the center of productivity (notes, tasks, journal, knowledge base)
- *Email* via mu4e with real-time sync
- *RSS* reading with Elfeed
- *AI assistance* via gptel (Claude)
- *Modern completion* with Corfu + Vertico + Consult

The config auto-switches between light/dark themes based on your system preference.

** First Steps
1. Run ~doom sync~ after any config changes
2. Run ~doom doctor~ to check for missing dependencies
3. Set up =~/.authinfo.gpg= with API keys (Anthropic, email, etc.)
4. Create =~/cloud/org/= directory structure
